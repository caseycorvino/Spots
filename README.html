Spotss

Spotss is an iOS application.

It is a follower based application where users add a spot to a map at their current location. The spot is then visible to all of their followers. 

App Features:
  - sign up
  - login
  - follower system
  - main feed page
  - search for users
  - profile page
  - selected user profile page
  - add a spot function
  - edit account
  - edit spots
  - temporary spots. the spots delete after 24 hours
